## Description

This project focuses on building an **intelligent NLP-based chatbot** capable of understanding user queries and generating relevant responses using Natural Language Processing techniques. The objective was to demonstrate **text processing, similarity matching, and conversational AI using Python**.

---

### 🔧 Project Implementation

The chatbot was developed using **Python in Visual Studio Code**, leveraging the following libraries:

* `nltk` – for text preprocessing (tokenization & lemmatization)
* `scikit-learn` – for TF-IDF vectorization and similarity computation
* `cosine_similarity` – for intent matching

The system follows an **intent-based architecture**, where each intent contains:

* User query patterns
* Intent tags
* Predefined responses

---

### ⚙️ Working Architecture

```text
User Input → Text Normalization → TF-IDF Vectorization → Cosine Similarity → Intent Matching → Response Generation
```

The chatbot processes user input by:

* Converting text to lowercase and removing punctuation
* Tokenizing and lemmatizing words using NLTK
* Transforming text into numerical vectors using TF-IDF
* Calculating similarity between user input and predefined patterns
* Selecting the most relevant intent and generating a response

A similarity threshold ensures that the chatbot responds only when confidence is sufficient, otherwise providing a fallback message.
